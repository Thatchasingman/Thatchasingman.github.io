xpath基础

简介

定义

​ xpath是一门在XML文档种查找信息的语言。

节点

在xpath，存在七种类型节点(node)。

  • 元素节点
  • 属性节点
  • 文本节点
  • 命名空间
  • 处理指令
  • 注释节点
  • 根节点

语法

[ / ] - - -> 从根节点选取

[ // ]- - ->从匹配选择的当前节点选择文档中的节点

[ . ]- - -> 选取当前节点

[ .. ] - - ->选取当前父节点

[ @ ] - - > 选取属性
